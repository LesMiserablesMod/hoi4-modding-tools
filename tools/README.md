# Tools

This directory is reserved for tools developed as part of this project.

Source code in this directory is licensed under **AGPL-3.0-or-later**, unless otherwise stated in a specific tool directory.

Possible future tools may include:

- Localisation checkers
- Focus tree validators
- Province ID checkers
- GFX reference checkers
- Mod template generators

The current priority of this project is to maintain a curated catalog of existing HOI4 modding tools and resources.

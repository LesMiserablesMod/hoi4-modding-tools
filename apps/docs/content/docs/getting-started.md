## Browse by workflow

Start with the catalog category that matches your current modding task:

- Map and province editing
- Focus tree creation
- Localisation editing
- GFX and asset work
- Scripting and effects
- Validation and linting
- Workshop and launcher workflows
- General references

## Evaluate a tool

Before adopting a tool, check:

- Whether the link and release page are still active.
- Which platform it supports.
- Whether it is maintained.
- Which license or usage terms apply.
- Whether it supports your HOI4 version and workflow.

## Suggest improvements

Use the GitHub issue templates to suggest a new tool, update existing information, report broken links, or suggest a category change.

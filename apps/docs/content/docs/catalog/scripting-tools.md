## Jomini

> A parser library for Clausewitz and Jomini data files.

- **Link**: https://github.com/nickbabcock/jomini
- **Repository**: https://github.com/nickbabcock/jomini
- **Platform**: Cross-platform
- **Type**: Library
- **Status**: Maintained
- **License**: MIT

### Use case

Use this when building custom tooling that needs to parse Clausewitz-style data files.

### Notes

This is a library rather than an end-user editor. Check upstream documentation for language bindings and supported formats.

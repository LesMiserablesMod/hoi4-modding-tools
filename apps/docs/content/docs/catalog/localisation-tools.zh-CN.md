## Paradox Translation Toolkit

> 面向 P 社游戏的本地化文件管理工具。

- **链接**: https://github.com/khoeos/paradox-translation-toolkit/
- **仓库**: https://github.com/khoeos/paradox-translation-toolkit/
- **平台**: Windows
- **类型**: 本地化工具
- **状态**: 活跃开发
- **许可证**: CC BY-NC-SA 4.0
- **语言**: TypeScript

### 使用场景

用于根据已有源语言，为 P 社游戏 Mod 生成缺失的目标语言本地化文件。

### 备注

上游项目说明其主要使用 Stellaris 测试，并计划支持多个 P 社游戏。用于 HOI4 工作流前，请先确认兼容性和发布说明。

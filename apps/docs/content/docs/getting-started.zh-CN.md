## 按工作流浏览

先从与你当前任务匹配的分类开始：

- 地图与省份编辑
- 国策树制作
- 本地化编辑
- GFX 与美术资源处理
- 脚本、效果与触发器
- 校验与 Lint
- 创意工坊与启动器工作流
- 通用参考资料

## 评估一个工具

采用某个工具前，建议先检查：

- 链接和发布页是否仍然可用。
- 支持哪些平台。
- 是否仍在维护。
- 使用什么许可证或条款。
- 是否适合你的 HOI4 版本和工作流。

## 提交改进

你可以使用 GitHub Issue 模板推荐新工具、更新已有信息、报告失效链接，或建议分类调整。

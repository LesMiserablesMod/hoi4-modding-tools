Use the following format when adding a new tool to the catalog.

## Tool name

> One-sentence description of what the tool does.

- **Link**: https://example.com
- **Repository**: https://github.com/example/example
- **Category**: Map / Focus Tree / Localisation / GFX / Scripting / Validation / Workshop / General
- **Platform**: Windows / Linux / macOS / Web
- **Type**: Editor / Validator / Converter / Script / Library / Reference
- **Status**: Active / Maintained / Unmaintained / Unknown
- **License**: MIT / GPL / Apache-2.0 / Proprietary / Unknown
- **Language**: Python / JavaScript / TypeScript / C# / Rust / Unknown

## Use case

Explain what problem this tool solves for HOI4 modders.

Mention limitations, compatibility issues, setup notes, or anything users should know before using it.

---
title: Contribute
description: How to contribute tools, documentation, and source code to HOI4 Modding Tools.
---

Contributions are welcome.

You can help by:

- Suggesting a new HOI4 modding tool.
- Updating outdated tool information.
- Reporting broken links.
- Improving descriptions, categories, and usage notes.
- Contributing tools developed under this project.

Before submitting a pull request, read the tool entry format, writing style, and license policy.
